# Personas

## Introdução


&emsp;&emsp;As personas são personagem fictício e hipotético, que é constituido por um grupo de usuários reais, com o objetivo de descrver um usuário so sistema. (Cooper et al., 2007; Pruitt e Adlin,
2006; Cooper, 1999).
